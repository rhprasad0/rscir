This is my implementation of the image + text search algorithm described in Psomas et al's paper.
Please check out their fantastic work here.
This implementation uses image embeddings from the PatterNet dataset thrown at the large RemoteCLIP model.
The composite approach in the paper had the highest accuracy with lambda=0.6, which is what is implemented here.
